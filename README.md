# HW1 - 個人資訊與即時時間網頁

一個精心設計的單頁網頁 (Single Page Application)，展示個人基本資訊並提供即時更新的數位時鐘與日期顯示。

🔗 **[線上預覽 / Live Demo](https://kevinlin13.github.io/hw1/)**

## 🌟 特色功能

- **精美視覺設計 (Premium Aesthetics)**
  - 採用現代的**毛玻璃 (Glassmorphism)** 卡片風格，具備高質感的半透明模糊背景與細緻邊框。
  - 動態**漸變背景光暈** (Gradient Mesh Glow)，在背景緩慢漂浮與縮放，營造流暢的視覺深度。
  - 名字「**林建璋**」採用漸層文字效果（紫、粉、藍），極具現代感。
- **即時數位時鐘 (Live Clock)**
  - 每秒自動更新時間（時、分、秒），搭配閃爍的冒號分隔符。
  - 使用 Google Fonts 的 `JetBrains Mono` 字型，確保時鐘數字等寬對齊、不晃動。
  - 顯示詳細的本地日期與星期（符合台灣習慣格式）。
  - 自動偵測並顯示當前時區標籤（如 `GMT+8`）。
- **響應式設計 (Responsive Design)**
  - 完美適應各式螢幕尺寸，包含手機、平板與桌上型電腦。

## 🛠️ 技術棧

- **Markup:** HTML5 (語意化標籤)
- **Styling:** Vanilla CSS3 (自訂 CSS 變數、Flexbox、Keyframe 動態特效、Backdrop Filter)
- **Logic:** Native JavaScript (DOM 操作、`setInterval` 即時同步更新)
- **Fonts:** Google Fonts (Outfit, Noto Sans TC, JetBrains Mono)

## 🚀 如何運行

1. 將此儲存庫複製 (Clone) 到本地，或直接下載 [index.html](index.html)。
2. 在您的電腦上雙擊 [index.html](index.html)，即可直接於瀏覽器中開啟並預覽。
